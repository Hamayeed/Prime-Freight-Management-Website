# PFM Truck Dispatch Service

A professional, responsive, and modular website for a Truck Dispatch Service.

## Architecture & Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES Modules)
- **Styling**: Vanilla CSS (Variables, Component-based CSS)
- **Build Tool**: Vite (for fast development and bundled production build)
- **Architecture**: Component-based architecture for scalability and maintainability.

## Folder Structure Highlights
- `/src/components`: Reusable UI elements and layout pieces.
- `/src/pages`: Distinct views representing website pages.
- `/src/styles`: Modular CSS focusing on reusability and theming.

## How to Run

1. Install Dependencies:
   ```bash
   npm install
   ```
2. Start Development Server:
   ```bash
   npm run dev
   ```
3. Build for Production:
   ```bash
   npm run build
   ```
